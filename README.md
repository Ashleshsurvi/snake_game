# 🐍 Futuristic Nokia Snake Game

A web-based remake of the classic Nokia Snake Game with a modern futuristic UI.

---

## 🚀 Features

- Classic Snake gameplay
- Modern, responsive, futuristic UI
- Simple and lightweight setup
